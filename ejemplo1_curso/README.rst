================

Este módulo extiende la funcionalidad de la vista "Albaranes a facturar" con el objetivo de permitir la agrupación por cliente en la búsqueda.


Instalación
===========

Simplemente instalar el módulo.

Configuración
=============

No hay nada que configurar.

Uso
===

Para utilizar este módulo, necesitas:

#. Ir a Ventas/Contabilidad/Albaranes a facturar.
#. A la derecha de la barra de búsqueda, pulsar la flecha de búsqueda avanzada.
#. Seleccionar "Cliente" en "Agrupar por"
