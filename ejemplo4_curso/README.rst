================
{ejemplo4_curso}
================

Este módulo modifica la factura de venta para adaptarla a las necesidades de la empresa.

Instalación
===========

Simplemente instalar el módulo.

Configuración
=============

No hay nada que configurar.

Uso
===

Para utilizar este módulo, necesitas:

#. Ir a Contabilidad/Clientes/Facturas de clientes.
#. Abrir una factura existente o crear una nueva.
#. Imprimir la factura para ver el nuevo informe.
