================
{ejemplo2_curso}
================

Este módulo extiende la funcionalidad del Presupuesto/Pedido de venta con el objetivo de añadir una nueva pestaña para las ofertas, así como el nuevo informe de oferta.


Instalación
===========

Simplemente instalar el módulo.

Configuración
=============

No hay nada que configurar.

Uso
===

Para utilizar este módulo, necesitas:

#. Ir a Ventas/Ventas/Pedidos de ventas.
#. Abrir un pedido existente o crear uno nuevo.
#. Pulsar en la pestaña Ofertas.
